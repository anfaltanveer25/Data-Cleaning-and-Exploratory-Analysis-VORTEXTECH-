# Wholesale Customers Data Analysis

## 📌 Project Overview

This project demonstrates the complete data preprocessing and exploratory data analysis (EDA) workflow using the Wholesale Customers dataset in Python. The objective is to clean the dataset, perform statistical analysis, and visualize customer purchasing behavior through informative charts.

## 🚀 Features

- Loaded and explored the dataset using Pandas
- Checked for missing values and duplicate records
- Handled missing values in numerical features
- Corrected data types where necessary
- Generated descriptive statistics
- Visualized data distributions using histograms
- Added appropriate titles and axis labels for all visualizations

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Dataset

The project uses the **Wholesale Customers Dataset**, which contains annual spending data of wholesale customers across different product categories, including:

- Fresh
- Milk
- Grocery
- Frozen
- Detergents_Paper
- Delicassen
- Channel
- Region
